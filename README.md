# NewMind

[中文](#中文) | [English](#english)

## 中文

NewMind 是一个给 AI 重度用户准备的本地记忆助手。

如果你经常使用 Codex、Claude、ChatGPT 或其他 AI Agent 做项目，可能会遇到这些问题：

- 每次开新对话，都要重新解释项目背景
- AI 忘记之前做过什么、决定过什么
- 长线程越来越卡，但旧上下文又舍不得丢
- 换一个 Agent 后，所有上下文又要重新讲一遍
- Codex 长期使用后，本地会话、日志和缓存越来越大，影响流畅度

NewMind 想解决的就是这些事。

它会帮你把项目背景、重要决定、工作进度、偏好和可复用经验保存成可继续使用的本地记忆，让 AI 工具更容易接上你的工作。

## NewMind 能做什么

### 让 AI 记住项目

NewMind 可以保存项目背景、当前进度、重要决定和常用规则，减少你反复解释的时间。

### 让长项目更容易继续

当一个项目跨越很多天、很多会话时，NewMind 可以帮助整理关键上下文，让新的 AI 对话更快进入状态。

### 帮 Codex 保持流畅

NewMind 提供 Codex 瘦身与维护能力，可以帮助清理和归档长期使用后膨胀的会话、日志和本地数据，减少卡顿和上下文压力。

### 管理你的 AI 记忆

你可以查看、纠正、替换或废弃旧记忆，避免错误信息一直影响后续工作。

### 本地优先

NewMind 面向个人本地工作流设计。你的长期记忆优先保存在本地，由你控制。

### 面向多 Agent 的未来

NewMind 当前重点支持 Codex 风格工作流。后续目标是适配更多 AI Agent，让不同 Agent 可以共享同一套项目记忆，并尽可能实现无缝切换。

## 适合谁

- 使用 Codex 时遇到长线程、缓存膨胀或卡顿问题的人

## English

NewMind is a local memory assistant for heavy AI users.

If you often use Codex, Claude, ChatGPT, or other AI agents for real projects, you may run into these problems:

- You need to explain the same project background again and again
- The AI forgets previous decisions and progress
- Long sessions become slower, but old context is still valuable
- Switching to another agent means starting from scratch
- Codex local sessions, logs, and cache can grow over time and affect smoothness

NewMind is built to help with these problems.

It keeps project background, decisions, progress, preferences, and reusable knowledge as local memory, so AI tools can continue your work with less repeated explanation.

## What NewMind Can Do

### Help AI Remember Your Projects

NewMind keeps project background, current progress, decisions, and useful rules, so you spend less time repeating yourself.

### Continue Long-Running Work

For projects that span many days and many sessions, NewMind helps preserve the important context for the next AI conversation.

### Keep Codex Running Smoothly

NewMind includes Codex slimming and maintenance tools to help archive and clean up large local sessions, logs, and data that build up over time.

### Manage AI Memory

You can review, correct, replace, or retire memories so outdated information does not keep affecting future work.

### Local-First

NewMind is designed for personal local workflows. Your long-term memory is stored locally and stays under your control.

### Future Multi-Agent Support

NewMind currently focuses on Codex-style workflows. The long-term goal is to support more AI agents and make it possible to switch between agents while keeping the same project memory.

## Who It Is For

NewMind is useful if you:

- Use AI for coding, games, tools, writing, or creative projects
- Do not want to explain the same project background every time
- Want AI to remember your preferences and decisions over time
- Run into long-session, cache growth, or slowdown issues with Codex
- Want to switch between different AI agents without losing context

## Current Status

NewMind is an early public release and is still being improved.

Current focus:

- Windows installer
- Local-first personal memory workflow
- Codex-style AI workflow support
- Codex slimming and maintenance
- Project memory, session continuity, and context recovery

Future plans:

- Support more AI agents
- Let different agents share the same project memory
- Make switching between agents as seamless as possible
- Improve memory management, automatic organization, and visual browsing
