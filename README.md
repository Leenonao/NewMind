# NewMind

NewMind is a local AI context and memory assistant for Codex-style workflows.

## Download

Download the latest Windows version here:

[Download NewMind](https://github.com/Leenonao/NewMind/releases/latest)

## What It Does

- Keeps compact project and session context
- Helps AI coding tools continue work across sessions
- Runs locally
- Designed for personal AI workflow management
